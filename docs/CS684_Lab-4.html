CS684 Spring 2022

CS684: Embedded System Course

Lab 4: Device Drivers for IO, Motor, PWM and Interrupts

In this lab, you have to write the device drivers of the project you have chosen.

  • For Search and Rescue project, Color Sensor Interfacing and Pwm control using Sensor is to be implemented.
  • For Valet Parking project, IR Proximity Sensor Interfacing and Object Following is to be implemented.

Submission Instructions:

  • For Lab-4 submission you have to upload a .tar.gz file.

  • Folder should contain following:

    1. You can implement the logic using either Statechart or Heptagon.

    For Statechart:

    • Statechart Project folder: It should contain following
      • Logic.ysc file: containing the statechart implementation of the logic.
      • Logic.sgen file: for generating the c code from statechart.
      • Supervisor Folder: it contains supervisor.ino file and auto generated c files from the statechart.

    For Heptagon:

    • Heptagon Project folder: It should contain following
      • Logic.ept file: ontaining the heptagon implementation of the logic.
      • Integrate.sh file: to generate the c files from heptagon.
      • Supervisor Folder: it contains supervisor.ino file and auto generated c files from the heptagon.
    1. Readme.txt file:
      • Containing information about the assumptions made (number of sensors,etc.) and description about the conditions or the scenarios if considered.
      • Also add youtube link of the video - shoot the working of the robot based on the video instructions provided in respective project problem statement.
    2. Contribution.txt file: stating detailed contribution of each member

  • Compress the folder and rename it as <GroupName>_Lab_4.tar.gz.

  • Upload the file on Moodle.

  • There should be only one submission per group. Member having highest roll no should do the submission.
