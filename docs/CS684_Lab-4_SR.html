CS684: Embedded System Course

Lab 4: Device Drivers for IO, Motor, PWM and Interrupts

Aim

In this lab, you will be interfacing the color sensors and motors to the AlphaBot. Speed of the motors are controlled based on the color detected by the color sensors.

Description:

Interface the color sensors to the Bot. Depending on the color detected by the Sensor, speed of the AlphaBot should vary according to following table.

ColorSpeed of both motors
Red0
Green250
Blue125
No ColorPrevious value

Interfacing:

  1. Motors: On AlphBot, motors are connected to following pins:
Motor Driver PinArduino PinPort Pin
Direction-Left MotorIN1A0PC0
Direction-Left MotorIN2A1PC1
Direction-Right MotorIN3A2PC2
Direction-Right MotorIN4A4PC3
Enable-Left MotorENAD5PD5 (OC0B)
Enable-Right MotorENBD6PD6 (OC0A)

  • For direction - use Forward

  • For PWM - use 8 bit Fast PWM

  1. Color Sensor: You need to connect color sensors to the AlphaBot. You can connect the sensors to the shield pins provided on top PCB of the AlphaBot.

For details about the color sensor, go through this tutorial - Color Sesnor Interfacing

Programming Instructions:

  1. Motor interfacing and PWM initialisation code should be in Embedded C.
  2. Color sensor Interfacing code should be in Embedded C.
  3. Write the logic about velocity control of the motors based on sensor values in Statechart or Heptagon.
  4. Values received from color sensor should be given to the Statechart or Heptagon.
  5. Velocity values are output of Statechart or Heptagon

Note: Do not use Arduino functions for motor and color sensors. For debugging, you can use serial functions of arduino library. Interfacing codes need to be written using Embedded C only. Use masking wherever required.

Note: No marks will be given for Arduino codes. Marks will be deducted if masking is not used properly.

Video Instructions:

  • Put the AlphaBot on the platform. Make sure that both wheels are not touching the ground hence robot will be at stationary position.
  • Mount the color sensor on the AlphaBot keeping in mind Arena used for the Project.
  • Turn on the robot. Initially robot should move in forward direction with 255 velocity.
  • Keep different color chart papers in front of color sensor one by one and show the changes in velocities of the motors.
