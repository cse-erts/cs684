Software Specifications

Coppeliasim:

  • Teams need to use Eclipse, for writting code for Coppeliasim. The program is provided as Eclipse project.

  • Teams can create a new folder fot Project. Use the instructions as per previous labs.

  • As we are using multiple robots in the project, following file have been modified accordingly:

    • eBot_Sim_Predef.cpp

  • Teams can refer to following folder for the reference on how to use the functions in eBot_Sim_Predef.h file.

    • Reference_project.zip

  • For the project, scene is designed in coppeliasim. Download the following scene file:

    • Smart_Parking.ttt

  • For the sensing, actuation and communication, Embedded C should be used and for algorithm development, Heptagon should be used.

  • Use of any non-open source libraries is not allowed and will result in disqualification.

  • Teams are free to add any number of sensors in Coppeliasim scene for sensing the Empty, Reserved and Full parking slots.


Server:

  • Teams can use any language and open source libraries for creating an interface (desktop app or web app).
