Grafana: Data Visualization Tool

A part of the IoT solutions is analyzing all the data that is collected from sensors and actuators. Data in it's raw form is difficult to understand and also time consuming (imagine 10 million data points). Thus it becomes almost necessary to use techniques such as data visualization. Using data visualization, data becomes easy to understand and quick to explore and analyze. We will learn Grafana, a tool for analyzing time-series data.

Install Grafana on Debian or Ubuntu

Run the commands in the order given below in your terminal:

sudo apt-get install -y apt-transport-https
sudo apt-get install -y software-properties-common wget
wget -q -O - https://packages.grafana.com/gpg.key | sudo apt-key add -

echo "deb https://packages.grafana.com/oss/deb stable main" | sudo tee -a /etc/apt/sources.list.d/grafana.list

sudo apt-get update
sudo apt-get install grafana

If you are not using Debian or Ubuntu, instructions for other operating systems can be found here

Note: We are using OSS version of Grafana and not enterprise edition.

Starting Grafana

  • Here's how you would start Grafana on Debian / Ubuntu:

sudo service grafana-server start

  • Then in your browser, go to http://localhost:3000.

  • Enter username and password as admin and admin respectively.

Creating a Dashboard

Visualization for a project is managed under a dashboard. Creating a dashboard has 5 major steps. You can repeat the steps to add more dashboards.

  • Install any Plugins if required
  • Add Datasource
  • Create a new Dashboard
  • Add Panel under Dashboard
    • Add Query to select data
    • (Optional) Apply transforms
    • Select a visualization
    • (Optional) Customize options
  • Save and apply. That's it.

Install Plugin

  • Grafana doesn't have the functionality to visualize data from Google Spreadsheet.
  • The plugin Google Sheets provides this functionality.
  • Run the command shown below to install it sudo grafana-cli plugins install grafana-googlesheets-datasource
