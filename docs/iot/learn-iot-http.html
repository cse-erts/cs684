Example: CS684 Weather Station

The values of parameters Temperature, Humidity, Pressure and CO2 will be generated using random module in python. Then the data will be logged into Google Spreadsheet. Google App Script is used to send alert notification via e-mail on exceeding the Temperature above 25 and Grafana is used as a dashboard to visualize the data.

Database: Google Sheets

  • Google Spreadsheet is use as a database to directly push/log sensor data.
  • Google Sheets has upper limit of 5 Million rows. So, if you are logging sensor data at an interval of 1 minute then Google Sheets will allow you to log data for 9 years!!
  • Google Sheets is very powerful tool which can allow user to make sense out of their IoT Data which is also a crucial part of IoT.

Alert Notifications: Google Apps Scripting

  • Using Google Apps Script, we can automate and integrate solutions for extending the functionality of Google Applications. Apps Script makes use of cloud based scripting written in JavaScript, CSS or HTML for extending these features of Google Applications.
  • Google Apps Script attached to Google Sheets offers a facility of alerting the user via an e-mail if the sensor readings exceed a certain pre-defined threshold.

Data Visualization: Grafana

  • A Web based data visualization tool.
  • Has an opensource community edition and an enterprise extension.
  • Ability to visualize data from multiple sources.
  • Extensible with plugins.
  • Grafana is used at Stackoverow, Paytm, Digitalocean, Gojek, eBay and Tinder.
