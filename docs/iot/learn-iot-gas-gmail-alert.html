Sending Gmail Alerts

Aim

  • To write App script code which will send alert to specified gmail from data of google spreadsheet we created in previous example.

NOTE: Same spreadsheet which we created at Updating Data into Google Spreadsheets section can be used here also.

Steps

  • Create a new Sheet named "Emails" in the same spreadsheet and add gmail ids to which you want to send the notification. You can add multiple mail ids.

  • Open your App Script editor and add following code inside your doPost(e) function.

  • This code snippet checks the last logged data in the sheet, sends email if data is greater than the threshold we have given.

  • Follow the exact process we did for Updating Data into Google Spreadhseets
